# Firebase Miner (Chat App)
- Firebase Miner is a versatile chat application that offers secure and convenient user authentication, personalized profiles, and an engaging user interface with light and dark themes. The app is built with Firebase Authentication and provides real-time chat functionality.

## Features
### 1. Firebase Authentication
- Multiple Sign-In Options: Users can sign in using their Google account, as a guest, or with an email and password. This ensures a secure and personalized experience.
### 2. Profile Page
- User Profile Management: Users have access to a dedicated Profile Page where they can view and update their display name, profile picture, and other personal details.
### 3. Light & Dark Themes
- Theme Customization: The app offers light and dark themes, allowing users to choose a visual style that best suits their environment and preferences.
### 4. Splash Screen
- Welcoming Experience: Upon launching the app, users are greeted with a splash screen that features the app's logo, animations, or an engaging message, setting a positive tone for their experience.
### 5. Chat Page for Each User
Real-Time Messaging: The core of the app is the Chat Page, where users can engage in real-time conversations. The intuitive interface makes it easy to send and receive messages, ensuring smooth communication.
